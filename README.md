# ml-socialMedia-mentalHealth-relationship-prediction
Exploring the relationship between daily usage time on different social media platforms and mental health



## Dataset
Installing the dataset from [Kaddle](https://www.kaggle.com/datasets/ayeshaimran123/social-media-and-mental-health-balance/discussion?sort=undefined) via cURL:
```
#!/bin/bash
curl -L -o ~/Downloads/social-media-and-mental-health-balance.zip\
  https://www.kaggle.com/api/v1/datasets/download/ayeshaimran123/social-media-and-mental-health-balance

```

I have uploaded the csv file in data folder, please check the CSV dataset [here](https://raw.githubusercontent.com/EvaSuperL/ml-socialMedia-mentalHealth-relationship-prediction/refs/heads/main/data/Mental_Health_and_Social_Media_Balance_Dataset.csv).
